Ademco CID notes
https://peoplecall.com/dc05-ademco-contactid-codes

Qualifer is 1 = NEW event 3 = RESTORAL

CCCC Q EEE PP ZZZ

CCCC = 4 digit account number
Q = Qualifier (1 or 3)
EEE = Event Code
PP = Partition
ZZZ = Zone/User Number

Example:
1234 18 1131 01 015 8
System Account 1234 reports perimeter intrusion alarm on zone 15 of partition 1
1234 = Account
18 = Message Type
1131 = 1 (New Event), 131 (Perimeter alarm event code)
01 = Parition 1
015 = Zone 15
8 = Checksum
